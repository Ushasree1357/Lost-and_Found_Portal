Problem 12: Functional Requirements: 
FR-01: The system shall allow users to report lost items with details including item name, category, description, location, date, and image upload.
FR-02: The system shall allow users to report found items with details including item name, category, description, location found, date, and image upload.
FR-03: The system shall provide search and filtering functionality for users to find lost/found items by keywords, categories, location, and date ranges.
FR-04: The system shall support user authentication and registration with email verification and secure login.
FR-05: The system shall enable claim submission and management where users can claim found items and administrators can verify ownership.
FR-06: The system shall provide administrative functions for managing users, items, claims, and system configuration.
FR-07: The system shall include notification system to alert users about matches, claims, and status updates via email.
