Problem 20: CASE Tool Setup
Overview
For the SEAD project titled "Lost & Found Portal for Campus Efficiency", StarUML was selected as the CASE tool for system modelling.
Tool Selected
StarUML
Purpose
The tool was installed and configured to support the following UML diagrams:
•	Use Case Diagram
•	Class Diagram
•	Component Diagram
•	Deployment Diagram
Outcome
StarUML enabled structured system design and clear architectural visualization for the Lost & Found Portal.
