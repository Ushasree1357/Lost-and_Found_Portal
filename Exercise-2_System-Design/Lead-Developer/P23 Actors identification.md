Problem 23: Actors Identification
Overview
Primary system actors interacting with the Lost & Found Portal were identified.
Actors
•	Student
•	University Staff
•	Administrator
•	System
Responsibilities
Actor	Role
Student	Report lost/found items, search for items, submit claims
University Staff	Report lost/found items, assist in item recovery
Administrator	Manage users, verify claims, oversee system operations
System	Perform automated matching, send notifications, store data
Outcome
Clear user interaction flow was established for campus-wide lost-and-found operations.
